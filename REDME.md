# Student Grade Management System

## Project Purpose
The Student Grade Management System is designed to assist educational institutions in managing student records in a digital format. The application aims to streamline processes that were traditionally handled through paper records or spreadsheets, enabling teachers to efficiently store and manage student data.

## Features
- **Add Student**: Input details to add new student records, including Student ID, Name, and Marks.
- **Display All Students**: View a list of all students currently stored in the system.
- **Search for Student by ID**: Retrieve specific student details using their unique Student ID.
- **Calculate Average Marks**: Compute and display the average marks of all students in the system.
##How to run
